# Веб-приложение для обучения дошколят

## Обзор

Статический SPA на vanilla HTML/CSS/JS для подготовки детей 3–6 лет к школе. Хостинг на GitHub Pages. Без бэкенда, без регистрации — открыл и играешь.

**Направления:** математика, грамота, логика, мир вокруг, внимание/память.
**Режимы:** сюжетный (приключение с персонажем-совёнком) + свободный выбор мини-игр.

## Архитектура

### Структура проекта

```
kids/
├── index.html           — единственный HTML-файл
├── css/
│   └── styles.css       — стили + CSS-переменные для масштабирования
├── js/
│   ├── app.js           — точка входа, инициализация
│   ├── router.js        — навигация по хешам (#/menu, #/game/math-1)
│   ├── state.js         — прогресс ребёнка в localStorage
│   ├── engine.js        — движок мини-игр (рендерит задание по JSON-конфигу)
│   └── story.js         — сюжетный режим (последовательность + диалоги)
├── data/
│   ├── games.json       — каталог всех мини-игр с метаданными
│   └── levels/
│       ├── math.json
│       ├── literacy.json
│       ├── logic.json
│       ├── world.json
│       └── attention.json
└── assets/
    ├── images/
    └── sounds/
```

### Ключевые решения

- Роутинг на `hashchange` — без зависимостей
- Контент заданий в JSON — новое задание = новый объект в файле
- `localStorage` для прогресса
- ES-модули (`import/export`) — нативная поддержка браузерами, без сборщиков

## Экраны и навигация

```
[Главный экран] → [Выбор режима] → [Сюжетный режим]
                                  → [Свободный режим] → [Выбор темы] → [Задание]
```

1. **Главный экран** — персонаж-совёнок, кнопка "Играть", выбор возраста (3–4 / 5–6)
2. **Выбор режима** — "Приключение" (сюжетный) и "Выбери сам" (свободный)
3. **Свободный режим** — сетка тем с иконками и индикаторами прогресса (звёзды)
4. **Сюжетный режим** — карта-путь с точками заданий, диалоги между ними
5. **Экран задания** — задание на весь экран, прогресс сверху, кнопка "домой"

Кнопка "назад" всегда доступна. Из любого экрана можно вернуться на главный.

## Типы заданий

Движок рендерит задания по типу. Шесть типов покрывают все темы:

| Тип | Описание | Пример |
|-----|----------|--------|
| `choice` | Выбери правильный из 2–4 вариантов | "Сколько яблок?" |
| `drag-drop` | Перетащи элемент на место | "Собери слово из букв" |
| `match` | Соедини пары линиями | "Цифра — количество" |
| `sequence` | Расставь по порядку | "Что сначала, что потом?" |
| `find` | Найди на картинке | "Найди все круги" |
| `color` | Раскрась по заданию | "Раскрась треугольники красным" |

### Пример конфига задания

```json
{
  "id": "math-3",
  "type": "choice",
  "difficulty": 1,
  "question": "Сколько яблок?",
  "image": "apples-3.png",
  "options": ["2", "3", "4", "5"],
  "answer": "3"
}
```

## Адаптивная сложность и прогресс

### Уровни сложности

- **Уровень 1 (3–4 года)** — 2 варианта ответа, крупные картинки, счёт до 5, отдельные буквы
- **Уровень 2 (5–6 лет)** — 3–4 варианта, счёт до 20, слоги и слова, логические цепочки

### Адаптивный алгоритм

По последним 5 ответам в теме:
- 4–5 правильных → повышаем сложность
- 0–1 правильных → понижаем
- 2–3 → остаёмся

### Хранение прогресса (localStorage)

```json
{
  "age": 2,
  "stars": 12,
  "topics": {
    "math": { "completed": [1,2,3], "current": 4, "history": [1,1,0,1,1] },
    "literacy": { "completed": [1], "current": 2, "history": [1,0,1] }
  },
  "story": { "chapter": 2, "point": 3 }
}
```

### Мотивация

- Звёздочки за правильные ответы
- Каждые 10 звёзд — новая анимация/стикер совёнка
- Сюжетный режим — открытие новых локаций
- Никаких наказаний — при ошибке совёнок подбадривает и даёт подсказку

## UI/UX для дошколят

### Визуальные принципы

- Минимальный тап-зона — 60×60px
- Мягкая палитра (пастельные тона + яркие акценты)
- Скруглённые углы везде
- Максимум 4 интерактивных элемента на экране
- Иконки вместо текста где возможно

### Обратная связь

- Правильно — зелёная вспышка, весёлый звук, совёнок радуется
- Неправильно — мягкое покачивание, нейтральный звук, "Попробуй ещё раз!"
- Анимации 300–500ms

### Адаптивность

- Mobile-first (планшет/телефон родителя)
- Ландшафтная и портретная ориентации
- `--base-size` пересчитывается от `vmin`

### Доступность

- Крупные кнопки для неточного детского тапа
- Drag-and-drop дублируется тапом (выбрал → тапнул куда)
- Всё понятно без звука

## Контент MVP (~50 заданий)

### Математика (math.json)
- Счёт предметов (choice)
- Сравнение "где больше" (choice)
- Порядок цифр (sequence)
- Цифра — количество (match)
- Простое сложение для 5–6 лет (choice)

### Грамота (literacy.json)
- Узнай букву (choice)
- Найди букву на картинке (find)
- Собери слово из букв (drag-drop)
- Буква — картинка (match)
- Порядок слогов (sequence)

### Логика (logic.json)
- Продолжи ряд (choice)
- Что лишнее (choice)
- Последовательность событий (sequence)

### Мир вокруг (world.json)
- Времена года (choice/match)
- Животные и детёныши (match)
- Профессии (match)
- Цвета и формы (find/choice)

### Внимание и память (attention.json)
- Найди отличия (find)
- Запомни и найди (sequence)
